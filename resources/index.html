JQuery Samples

Zazl is a Dynamic Javascript Optimizer that enables the loading of AMD modules and all their dependencies in a single response stream without depending on static build tools. It also leverages JavaScript compression tools such as Google Closure, Uglify-js and Dojo Shrinksafe.

The samples use the Zazl AMD loader to load their modules.The Zazl backend component that performs all the dependency analysis and optimization is running on a Node.js based server.

The Text Messenger sample demonstrates the Zazl Optimizer loading a JQuery/JQuery mobile/Backbone/Underscore application. Take a look at the javascript network traffic in a browsers developers tools to see the optimization in action.
